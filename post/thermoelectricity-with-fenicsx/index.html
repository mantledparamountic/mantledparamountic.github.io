Thermoelectricity With FEniCSx

Isaac Haïk Dunn
Apr 23, 2023 9 min read 
'''
This program solves for the temperature distribution and the electrostatics potential in a wire in which an electric
current is passing. It computes the electric field as well as the thermal gradient, 
enabling the computation and plot of the electric and thermal fluxes. It contains the generalization with thermoelectricity,
which is enough to compute the efficiency of a device operating in a closed circuit.

Tested the code by setting a vanishing Seebeck coefficient and by selecting current i/o surface same as thermal problem.
This means I can trust my code and move on. Saved the file with ending "test_0_works.ipynb".

In this program, the "injected" current is the current produced by the voltage due to the temperature difference alone.
This corresponds to attaching a wire to the material (although in this case we should consider Peltier heat).

The implementation is as follows. First, the thermal problem (without current) is solved, which gives the voltage generated
by the temperature gradient. Then the thermoelectricity problem is solved iteratively, where there is an electric current caused
by the voltage of the previous iteration, until convergence in the solutions (temp and volt) is achieved. Dirichlet b.c. are used.


The first iteration works as expected. Tested with S=30e-6 V/K, I = 0 A. The code produces the correct E field, temperature, voltage, grad T, grad V, Je and JQ values.
'''

We initialize the temperature and voltage to reasonable values before we write down the weak form of the problem. We also specify some parameters to the solver.

from ufl import dot, derivative
from petsc4py import PETSc

def temp_init(x):
    values = np.full(x.shape[1], T_cold, dtype=PETSc.ScalarType) 
    return values

def volt_init(x):
    values = np.full(x.shape[1], V_current_contact_out, dtype=PETSc.ScalarType) 
    return values

# Initialize values for the temperature and voltage.
TempVolt.sub(0).interpolate(temp_init)
TempVolt.sub(1).interpolate(volt_init)

# Weak form.
J_vector = -sigma * grad(volt) - sigma * Seebeck_coeff * grad(temp)
Fourier_term = dot(-kappa * grad(temp), grad(u)) * dx
Joule_term = dot(rho * J_vector, J_vector) * u * dx

F_T = Fourier_term + Joule_term
F_V = -dot(grad(v), sigma * grad(volt))*dx -dot(grad(v), sigma * Seebeck_coeff * grad(temp))*dx #+ v * J * ds(vanish_voltage_surface) 
print('This should correspond to the current injected: ', assemble_scalar(form(J*ds(vanish_voltage_surface))))

weak_form = F_T + F_V

Jac = derivative(weak_form,TempVolt,dTV)

# Solve the PDE.
from dolfinx.fem.petsc import NonlinearProblem
from dolfinx.nls.petsc import NewtonSolver

problem = NonlinearProblem(weak_form, TempVolt, bcs=bcs, J = Jac)
solver = NewtonSolver(MPI.COMM_WORLD, problem)
solver.convergence_criterion = "incremental"
solver.rtol = 1e-14
solver.report = True


ksp = solver.krylov_solver
opts = PETSc.Options()
